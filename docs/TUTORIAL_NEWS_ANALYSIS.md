# 🚀 Быстрый старт: Анализ с Новостями

## 📋 Пошаговая инструкция

### Шаг 1: Загрузи данные
```bash
python main.py update
```
Система загружает/обновляет данные по всем акциям из вотчлиста.

### Шаг 2: Запусти анализ
```bash
python main.py analyze
```
Система генерирует markdown отчёт с техническим анализом.

### Шаг 3: Дай мне отчёт + CSV + новости

Просто скопируй и вставь последний отчёт:
```
Вот последний отчёт и данные. 
Новости если есть:
- SBER: ЦБ повысил ставку на 0.5%
- LEAS: Квартальный доход +15%
- X5: Открыл 3 новых магазина

Анализируй!
```

### Шаг 4: Я автоматически
```
1. 🔍 Ищу свежие новости в интернете по каждому BUY сигналу
2. 📰 Добавляю в контекст
3. ✅ Проверяю соответствие техсиналов и новостей
4. 📊 Создаю HTML отчёт с разделом "📰 Новостной фон"
```

### Шаг 5: Получишь результат
```
📊 RECOMMENDATIONS_20251115.html
├─ 📰 Новостной фон (если есть)
├─ 💡 Основание (с учётом новостей)
├─ 📈 Уровни и входы
├─ ⚠️ Риски (с учётом макро-событий)
└─ ✅ Вердикт (укреплен новостями или ослаблен)
```

## 📰 Примеры

### Пример 1: Новость подтверждает сигнал

**Техсигнал:** LEAS - BUY, восходящий тренд, RSI=60  
**Новость:** ТМК отчитался о выручке +15%, прибыль +20%  
**Результат:** ✅ **STRONG BUY** (сигнал усилен реальными результатами)

```html
📰 НОВОСТНОЙ ФОН: ТМК показала квартальный рост +15% 🟢
💡 ОСНОВАНИЕ: Техсигнал подтверждён реальными финрезультатами
✅ ВЫВОД: Рекомендуется вход на текущих уровнях
```

### Пример 2: Новость противоречит сигналу

**Техсигнал:** SBER - BUY, восходящий тренд  
**Новость:** ЦБ повысил ставку на 0.5%  
**Результат:** ⚠️ **BUY НА ОТКАТЕ** (добавляет риск)

```html
📰 НОВОСТНОЙ ФОН: ЦБ повысил ставку на 0.5% 🔴
⚠️ РИСКИ: Повышение ставок может давить на прибыль банков
💡 СОВЕТ: Входить осторожнее, увеличить стоп-лосс на 1%
```

### Пример 3: Неожиданная позитивная новость

**Техсигнал:** X5 - HOLD (нейтральный)  
**Новость:** X5 открыл 10 новых магазин (3x выше плана)  
**Результат:** 📈 **UPGRADE ДО BUY** (новость поменяла картину)

```html
📰 НОВОСТНОЙ ФОН: Агрессивная экспансия (10 магазин вместо 3) 🟢
💡 ОСНОВАНИЕ: Компания готова к росту, выше рыночные ожидания
✅ РЕКОМЕНДУЕМ: Переходим в BUY, точка входа консервативная
```

## 🎯 Когда новости помогают

| Сценарий | Техсигнал | Новость | Итог |
|----------|-----------|---------|------|
| Подтверждение | BUY | 🟢 Позитивная | ✅ STRONG BUY |
| Добавок риска | BUY | 🔴 Негативная | ⚠️ BUY НА ОТКАТЕ |
| Неожиданный апсайд | HOLD | 🟢 Очень позитивная | 📈 UPGRADE |
| Ослабление сигнала | SELL | 🟢 Позитивная | 🔄 HOLD (переосмыслить) |
| Подтверждение вниз | SELL | 🔴 Негативная | ❌ STRONG SELL |

## ⚡ Как это улучшает анализ

### Без новостей:
- ❌ Иногда техсигнал подсказывает рост, но компания теряет контракты
- ❌ Не видим макро-события (ставки ЦБ, санкции)
- ❌ Можем войти в "накопление" перед обвалом

### С новостями:
- ✅ Подтверждаем техсигналы реальными событиями
- ✅ Видим макро-риски заранее
- ✅ Ловим неожиданные возможности (компания растёт выше ожиданий)
- ✅ Уточняем время входа (откат после плохой новости часто даёт лучшую точку)

## 🔍 Что я ищу

Система ищет:
1. **Квартальные отчёты** - финансовые результаты
2. **Макро-события** - решения ЦБ, санкции, геополитика
3. **События компаний** - слияния, новые проекты, смена руководства
4. **События сектора** - регуляция, технологические сдвиги
5. **Аналитика** - обновления целевых цен от банков

## 💡 Советы

1. **Дай мне прямо скопированные новости**, если интернет медленный
2. **Используй официальные источники** - биржа, РИА, ТАСС
3. **Актуальность важна** - новость из 3х месяцев назад можно не скидывать
4. **По каждой новости указывай дату** - это помогает оценить влияние

Пример хорошей новости:
```
- 15 ноября 2025: ЦБ объявил повышение ставки на 0.5%
- 14 ноября: ТМК отчиталась о выручке 150млн (+15% YoY)
- 13 ноября: X5 открыт 10 новых магазин в регионах
```
